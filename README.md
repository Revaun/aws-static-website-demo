# AWS Static Website Demo

## Overview
This project demonstrates how to deploy a static website using **AWS S3**.  
It includes a simple HTML page (`index.html`) and shows the workflow of version control with Git and GitHub.

## Technologies Used
- **HTML5** for the static site
- **Git & GitHub** for version control and collaboration
- **AWS S3** (intended deployment target)
